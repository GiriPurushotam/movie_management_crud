 ** Movie Management System On Native PHP and Simple Ajax.

Project Name: 
* MovieDB Crud Web Application

Used Languages:
* PHP
* MYSQL
* CSS
* AJAX

Setup Instructions for localhost
* Create a database called "movie"
* Import the database.sql file into movie database to create tables and insert sample data
* Open the project in browser http://localhost/movie_project/public/index.php


Login creadentials:
* None

Features Implemented:
* Add, Edit, Delete Movies Records.
* Search movies with autocomplete.
* Flash messages for Add, Delete, Update movies.
* Display trending movies with casts and rating.
* XSS protection with htmlspecialchars().
* Separate header/footer, clean folder structure.


Known Issue:
* None
